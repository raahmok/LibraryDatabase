# Simulace databáze knihovny
CLI nastroj pro manipulaci s databázi virtualní knihovny

## Pouzite jazyky a frameworky
- C# => base aplikace
- SQLite => sql engine
- ADO.NET => komunikace s databazi

## Casova narocnost
- Nedele => 4 hodiny
- Pondeli => 4 hodiny
- Utery => 7 hodin
- ### celkem 15 hodin

## Features
- katalog knih
- sprava uzivatelu
- pujcovani a vraceni knih
- admin privilegia
- log vypujcek
- upravovani tabulek
